# Lab 3: Rust/C FFI and Unsafe Boundaries

## Overview
Students inspect a Rust wrapper around a C function and identify where
safety depends on an external contract.

## Key Concepts
- Rust unsafe code
- FFI boundaries
- ABI and calling conventions
- Type-safety across language boundaries

## Prerequisites
- Basic Rust understanding
- Ghidra familiarity

##  Expected Deliverables
1. Location of the call boundary in binary
2. Argument passing ABI analysis
3. Explanation of contract violation
4. Mitigation proposal
