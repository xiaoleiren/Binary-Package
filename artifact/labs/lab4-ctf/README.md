# Lab 4: Integrated CTF Challenge

## Overview
The final binary combines one pattern from each earlier lab. Students must
reason across source, binary, and execution trace to find the flag.

## Key Concepts
- Synthesis across optimization, UB, and FFI
- Attack chain construction
- Evidence-based reasoning

## Prerequisites
- All prior labs completed
- Ghidra and GDB proficiency

## Expected Deliverables
1. Flag
2. GDB trace showing key breakpoints
3. Ghidra annotations
4. Source-to-binary reasoning report (explaining each pattern)
5. Mitigation proposal
